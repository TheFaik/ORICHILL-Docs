# Подключение c Raknetify

Если в вашем клиенте имеется мод [**Raknetify**](https://modrinth.com/plugin/raknetify), то вы можете воспользоваться им для подключения к нашему серверу.

Вот IP-Адреса для этого:

* `raknet;mc.orichill.ru`
* `raknetl;mc.orichill.ru`

Вставьте их, как обычный IP-Адрес и подключайтесь.<br>
