# Защита приватизированной территории

{% hint style="danger" %}
<kbd><mark style="color:red;">**‼️ ВНИМАНИЕ ‼️**<mark style="color:red;"></kbd>\
<kbd><mark style="color:red;">↳<mark style="color:red;"></kbd> _**Мы только начали работу над интеграцией плагина&#x20;**<mark style="color:$info;">**`WorldGuard`**</mark>**&#x20;в нашу экосистему/эргономику, а потому не весь&#x20;**<mark style="color:$primary;">**`WIKI`**</mark>_ _**и сервер подготовлены к его использованию.**_\
_**В рамках того, что у нас сервер находится в стадии&#x20;**<mark style="color:$warning;">**`DEV`**</mark>**&#x20;- мы постепенно решим данную проблему.**_ \
\
**Следите за новостями 💕**
:::

### <mark style="color:$primary;">**ИНФОРМАЦИЯ ПО ПЛАГИНУ И ЕГО ХАРАКТИРИСТИКА**</mark>

{% hint style="info" icon="readme" %}
<kbd>**Если вкратце:**</kbd> _плагин мы используем, чтобы игроки смогли защитить свою территорию от несанкционированного использования третьими лицами (особенно от недобродушных новичков)._\
<kbd>**^Примечание:**</kbd> _Создание регионов не помешает другим людям открывать сундуки/двери/люки, пользоваться вагонетками и т.п._ \
\
_Регионы были введены для защиты ваших строений, а для защиты всего вышеперечисленного используйте информацию из этого раздела -_ [protect-chest.md](protect-chest.md "mention")\
\
_\*Введено это было по опыту работы сервера в период конца 2023 по октябрь 2025 года._ \
_^Увы, но игроки бывают разные, поэтому, чтобы не мучить добродушных игроков - мы ввели данный функционал для защиты собственности._
:::

### <mark style="color:$primary;">**Ваши ограничения использования команд в WorldGuard:**</mark>

{% hint style="success" icon="shield-check" %}
<mark style="color:$success;">**Регионы можно создать во всех мирах**</mark>
:::

* Разрешается до <mark style="color:$primary;">**`7`**</mark> регионов;
* Количество блоков для привата - <mark style="color:$primary;">**`2 млн`**</mark>;
* До <mark style="color:$primary;">**`300`**</mark> сущностей в каждом регионе.

{% hint style="info" icon="tag" %}
Если вам вдруг понадобится больше - обращайтесь к Администрации сервера через открытие тикета на нашем сервере Discord. <br>

<kbd><mark style="color:$success;">**^Рекомендуется**<mark style="color:$success;"></kbd> объяснить причину, по которой вам понадобилось больше количества регионов/блоков привата/cущностей в регионе.
:::
