  <kbd>_**^Пояснение:**_</kbd> Для усложнения игрового процесса. Теперь, чтобы узнать координаты, надо иметь компас.&#x20;

* <kbd>**‣**</kbd>**&#x20;Изменение в игровой процесс:** Укрепленный Глубокий Сланец теперь падает при добыче инструментом с шелковистым прикосновением.
* <kbd>**‣**</kbd> Отключение надписи в **Наковальне** и в **Столе Зачарования** "**Очень Дорого**".
* <kbd>**‣**</kbd> Фантомы стали **Дроном-Камикадзе**.&#x20;
* <kbd>**‣**</kbd> Добавлена возможность для **Тотема Бессмертия**, которая теперь может спасти игрока от смерти **в пустоте**, телепортируя в последнее место, где его ноги коснулись земли. Если по какой-либо причине эта позиция не будет найдена, они будут телепортированы на точку возрождения.
* <kbd>**‣**</kbd> Игроки могут возвращать чары предмета в виде книги с помощью наковальни за 25 уровней опыта.
* <kbd>**‣**</kbd> Деревянные рецепты для камнереза.
