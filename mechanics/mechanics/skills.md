## <mark style="color:$primary;">Навыки</mark>

Навыки повышаются путем выполнения определенных задач и дают эффекты, характеристики и другие бонусы, при повышении уровня.\
\
<kbd>**Всего навыков 9, а именно:**</kbd>

| Навык                                                                                       | Задача                                        |
| ------------------------------------------------------------------------------------------- | --------------------------------------------- |
| `⛏️` <kbd><mark style="color:$primary;">**Шахтёрство**<mark style="color:$primary;"></kbd>  | _Разбивайте блоки, чтобы заработать XP_       |
| `⚔️` <kbd><mark style="color:$info;">**Боевой**<mark style="color:$info;"></kbd>            | _Убивайте мобов, чтобы заработать XP_         |
| `🔮` <kbd><mark style="color:$success;">**Зачарование**<mark style="color:$success;"></kbd> | _Зачаровывайте предметы, чтобы заработать XP_ |
| `🌾` <kbd><mark style="color:$warning;">**Фермерство**<mark style="color:$warning;"></kbd>  | _Собирайте урожай, чтобы заработать XP._      |
| `🪓` <kbd><mark style="color:$danger;">**Лесорубство**<mark style="color:$danger;"></kbd>   | _Рубите деревья, чтобы заработать XP_         |
| `🎣` <kbd><mark style="color:blue;">**Рыболовство**<mark style="color:blue;"></kbd>         | _Рыбачьте, чтобы заработать XP_               |
| `🧪` <kbd><mark style="color:purple;">**Алхимия**<mark style="color:purple;"></kbd>         | _Варите зелья, чтобы заработать XP_           |
| `🩻` <kbd><mark style="color:orange;">**Мазохизм**<mark style="color:orange;"></kbd>        | _Получайте урон, чтобы заработать XP_         |
| `🕶️` <kbd><mark style="color:green;">**Разведка**<mark style="color:green;"></kbd>         | _Двигайтесь, чтобы заработать XP_             |
