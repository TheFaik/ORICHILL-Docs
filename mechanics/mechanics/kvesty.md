# Квесты

::: info
После каждого прохождения квеста, он будет доступен через <kbd>_<mark style="color:$primary;">**24 часа**</mark>_</kbd>.
:::

## <mark style="color:$primary;">Информация про квесты</mark>

На нашем любимом сервере присутствуют квесты. \
Они помогут улучшить геймплей для игроков, которые не могут придумать себе цель, да и с помощью квестов, можно быстрее прокачиваться по навыкам.\
\
**На данный момент всего 5 категорий квестов, а именно:**

* <kbd><mark style="color:green;">**⛏️**<mark style="color:green;"></kbd> <mark style="color:$info;">**Добыча руд**</mark>
* <kbd><mark style="color:purple;">**🧙‍♂️**<mark style="color:purple;"></kbd> <mark style="color:$primary;">**Чарующий квест**</mark>
* <kbd><mark style="color:blue;">**🔬**<mark style="color:blue;"></kbd> <mark style="color:orange;">**Исследование**</mark>
* <kbd><mark style="color:yellow;">**🧑‍🌾**<mark style="color:yellow;"></kbd> <mark style="color:$success;">**Фермерство**</mark>
* <kbd><mark style="color:orange;">**🎣**<mark style="color:orange;"></kbd> <mark style="color:blue;">**Рыболовство**</mark>

{% hint style="info" icon="square-info" %}
Для просмотра квестов, используйте команду: <kbd><mark style="color:$primary;">**/quest**<mark style="color:$primary;"></kbd>, либо через меню сервера: <kbd><mark style="color:$primary;">**/menu**<mark style="color:$primary;"></kbd>
:::
