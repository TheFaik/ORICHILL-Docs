# Декоративные Модели

{% embed url="https://youtu.be/jaNsdR6UBNI" %}

## <mark style="color:$primary;">Использование</mark>

Для использования, вам, как минимум, нужна **`Стока для Брони`**, после чего вводим команду \
<kbd><mark style="color:$primary;">**/eas give**<mark style="color:$primary;"></kbd>, чтобы получить **`Редактор (EAS)`**&#x438; начинаем редактировать Модель.

## <mark style="color:$primary;">Команды</mark>

<mark style="color:$primary;">**`/eas align`**</mark> - переместить выбранную сущность в центр блока\
<mark style="color:$primary;">**`/eas cantick`**</mark> - вкл./выкл. тикание выбранной подставки для брони\
<mark style="color:$primary;">**`/eas clipboard`**</mark> - показать буфер обмена\
<mark style="color:$primary;">**`/eas copy`**</mark> - скопировать все свойства выбранной сущности\
<mark style="color:$primary;">**`/eas give`**</mark> - выдать инструмент <mark style="color:$primary;">`Редактор (EAS)`</mark>\
<mark style="color:$primary;">**`/eas help`**</mark> - показать справку по командам\
<mark style="color:$primary;">**`/eas history`**</mark> - посмотреть историю\
<mark style="color:$primary;">**`/eas info`**</mark> - просмотреть подробную информацию о выбранной сущности\
<mark style="color:$primary;">**`/eas open`**</mark> - открыть меню\
<mark style="color:$primary;">**`/eas paste`**</mark> - применить буфер обмена к выбранной сущности\
<mark style="color:$primary;">**`/eas pitch`**</mark> - установить угол наклона выбранной сущности\
<mark style="color:$primary;">**`/eas position`**</mark> - телепортировать выбранную сущность\
<mark style="color:$primary;">**`/eas reset`**</mark> - сбросить значение выбранного инструмента\
<mark style="color:$primary;">**`/eas select`**</mark> - выбрать соответствующие сущности с помощью селектора\
<mark style="color:$primary;">**`/eas set`**</mark> - установить значение выбранного инструмента\
<mark style="color:$primary;">**`/eas snap angle`**</mark> - изменить шаг привязки угла\
<mark style="color:$primary;">**`/eas snap move`**</mark> - изменить шаг привязки позиции\
<mark style="color:$primary;">**`/eas yap`**</mark> - изменить направление взгляда выбранной сущности
